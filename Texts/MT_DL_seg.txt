触覚 刺激 が ない 場合 の 触覚 ゲーティング の 予期 せ ぬ 現象学 的 効果 で ある ファントム ・ タッチ 錯視 
我々 は 、 物理 的 刺激 を 伴わ ない 自己 接触 時 に 知覚 さ れる ピリピリ 感 の 存在 を 報告 する 。 
我々 は 没入 型 バーチャルリアリティシナリオ を 用い 、 被験 者 は バーチャルオブジェクト を 用い て 自分 の 身体 に 触れ た 。 
この タッチ は 、 仮想 身体 上 の タッチ さ れ た 場所 に 対応 する ピリピリ し た 感覚 を も たらし た 。 
われわれ は これ を 「 ファントム ・ タッチ  錯視 」 （ PTI ） と 呼ん だ 。 
興味 深い こと に 、 この 錯視 は 、 被験 者 が 自分 の 手足 の 見え ない （ 推測 さ れる ） 部分 を 触っ た とき に も 報告 さ れ た 。 
我々 は 、 この PTI は 、 抑制 す べき 触覚 入力 が ない 場合 に 、 自己 触覚 時 に 触覚 ゲーティング が 起こる ため に 起こる と 考え た 。 
目 に 見え ない 身体 の 一部 を 触っ た とき に 報告 さ れ た PTI は 、 触覚 ゲーティング が 視覚 のみ に 基づく の で は なく 、 むしろ 身体 スキーマ を 含む 多感覚 的 な トップダウン 入力 に 基づく こと を 示し て いる 。 
この 補足 的 な 発見 は 、 自分 の 身体 に 関する 表象 は 、 利用 可能 な 感覚 情報 を 超え て トップダウン で 定義 さ れる こと を 示し て いる 。
