ファントムタッチ 錯覚 触覚 刺激 が ない 場合 の 触覚 ゲート の 現象論 的 効果 
本研究 で は 、 物理 的 刺激 を 伴わ ない 自己 接触 中 に 知覚 さ れる チクチク 感 の 存在 を 報告 する 。 
我々 は 、 参加 者 が 仮想 オブジェクト を 使用 し て 自分 の 体 に 触れる 没入 型 仮想 現実 シナリオ を 使用 し た 。 
この 接触 により 、 仮想 身体 上 で 触れ られ た 位置 に 対応 する 部位 で チクチク する 感覚 が 生じ た 。 
我々 は これ を 「 ファントム ・ タッチ ・ イリュージョン 」 （ PTI ） と 名付け た 。 
興味 深い こと に 、 参加 者 が 手足 の 目 に 見え ない （ しかし 推測 さ れる ） 部分 に 触れ た とき の 錯覚 も 報告 さ れ た 。 
この PTI は 、 抑制 する 触覚 入力 が ない 場合 の 自己 接触 中 の 触覚 ゲート | プロセス の 結果 で ある と 考え られ ます 。 
目 に 見え ない 身体 部分 に 触れ た とき に 報告 さ れ た PTI は 、 触覚 ゲート が 視覚 のみ に 基づい て いる の で は なく 、 身体 スキーマ を 含む 多感 覚 の トップダウン 入力 に 基づい て いる こと を 示し て いる 。 
この 結果 は 、 身体 表象 が 、 利用 可能 な 感覚 情報 を 超え て 、 トップダウン で 定義 さ れる こと を 示し て いる 。
