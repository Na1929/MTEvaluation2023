触覚 刺激 が ない 場合 の 触覚 ゲート の 予期 せ ぬ 現象論 的 効果 で ある ファントム ・ タッチ 錯視  
本研究 で は 、 物理 的 刺激 を 伴わ ない 自己 接触 時 に 知覚 さ れる ピリピリ 感 の 存在 を 報告 する 。 
我々 は 没入 型 仮想 現実 シナリオ を 用い 、 参加 者 は 仮想 オブジェクト を 用い て 自分 の 身体 に 触れ た 。 
この 接触 は 、 仮想 身体 上 で 触れ られ た 場所 に 対応 する 部位 に ピリピリ し た 感覚 を も たらし た 。 
われわれ は これ を 「 ファントム ・ タッチ 錯視 」 （ PTI ） と 名付け た 。 
興味 深い こと に 、 この 錯視 は 、 参加 者 が 手足 の 見え ない （ しかし 推測 さ れる ） 部分 を 触っ た とき に も 報告 さ れ た 。 
我々 は 、 この PTI は 、 抑制 す べき 触覚 入力 が ない 場合 に 、  自己 触覚 時 の 触覚 ゲート 処理 から 起こる と 考え た 。 
目 に 見え ない 身体 の 一部 を 触っ た とき に 報告 さ れ た PTI は 、 触覚 ゲート が 視覚 のみ に 基づく の で は なく 、 むしろ 身体 スキーマ を 含む 多感 覚的 な トップダウン 入力 に 基づく こと を 示し て いる 。 
この 結果 は 、 身体 表象 は 、 利用 可能 な 感覚 情報 を 超え て トップダウン で 定義 さ れる こと を 示し て いる 。
