ファントムタッチ 錯覚 、 触覚 刺激 が ない 場合 の 触覚 ゲート の 予期 せ ぬ 現象学 的 効果 
我々 は 、 物理 的 刺激 を 伴わ ない セルフタッチ 中 に 知覚 さ れる チクチク 感 の 存在 を 報告 する 。
私 たち は 、 被験 者 が 仮想 オブジェクト を 使用 し て 自分 の 体 に 触れる 没入 型 仮想 現実 シナリオ を 使用 し まし た 。 
この タッチ により 、 仮想 ボディ 上 の タッチ さ れ た 位置 に 対応 する チクチク する 感覚 が 生じ まし た 。 
私 たち は これ を 「 ファントム ・ タッチ ・ イリュージョン 」 （ PTI ） と 呼び まし た 。 
興味 深い こと に 、 被験 者 が 手足 の 目 に 見え ない （ 推測 さ れる ） 部分 に 触れ た とき の 錯覚 も 報告 さ れ まし た 。 
この PTI は 、 抑制 する 触覚 入力 が ない 場合 の セルフタッチ 中 の 触覚 ゲート プロセス の 結果 で ある と 考え られ ます 。 
目 に 見え ない 身体 部分 に 触れ た とき に 報告 さ れ た PTI は 、 触覚 ゲート が 視覚 のみ に 基づい て いる の で は なく 、 身体 スキーマ を 含む 多感 覚 の トップダウン 入力 に 基づい て いる こと を 示し て い ます 。 
この 補足 的 な 発見 は 、 自分 自身 の 身体 の 表現 が 、 利用 可能 な 感覚 情報 を 超え て 、 トップダウン で 定義 さ れる こと を 示し て い ます 。
